# Projeto Jokempo em C

## Descrição do Projeto

O projeto foi desenvolvido como atividade acadêmica com foco na aplicação prática dos conteúdos estudados em sala de aula.

---

## Funcionalidades Implementadas

O sistema possui as seguintes funcionalidades:

- Modularização do código por meio da criação de métodos/funções;
- Validação das entradas de dados fornecidas pelo usuário;
- Cadastro e gravação do nome do jogador;
- Opção para alterar o jogador durante a execução do programa;
- Exibição de estatísticas dos jogadores;
- Execução do jogo Jokempo de forma interativa no terminal.
